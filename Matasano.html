chunk(string, n)
(function) # ↑

Specs:

  • chunk(String.t, non_neg_integer) :: [String.t]

Returns a list containing n strings, where each new chunk starts n bytes into the string.

If there are not enough bytes to fill the final chunk, the partial chunk will be discarded from the result.

Examples

iex> Matasano.chunk("abcabcabc", 3)
["abc", "abc", "abc"]

iex> Matasano.chunk("xxxyyyzzz", 4)
["xxxy", "yyzz"]

repeating_xor(key, message)
(function) # ↑

Specs:

  • repeating_xor(iodata, iodata) :: binary

Performs bit-wise XOR (exclusive or) on the data supplied, repeating key as necessary to ensure equal length.

Examples

iex> Matasano.repeating_xor("a", "foobar")
<<7, 14, 14, 3, 0, 19>>
iex> Matasano.repeating_xor("a", <<7, 14, 14, 3, 0, 19>>)
"foobar"

iex> Matasano.repeating_xor("a", "hełło")
<<9, 4, 164, 227, 164, 227, 14>>
