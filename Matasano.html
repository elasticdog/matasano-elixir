character_frequency(string)
(function) # ↑

Specs:

  • character_frequency(String.t) :: Map

Counts the frequency of characters in string.

Returns a dictionary of characters mapping to count values. All characters are normalized to uppercase.

Examples

iex> Matasano.character_frequency("Hełło world!")
%{" " => 1, "!" => 1, "D" => 1, "E" => 1, "H" => 1, "L" => 1, "O" => 2,
  "R" => 1, "W" => 1, "Ł" => 2}

(function) # ↑

Specs:

  • language_score(String.t, Map) :: float

Calculates a score based on the probability of string being text of a particular language.

The higher the score, the higher the likelihood of the text being written in the language as defined by the given language_distribution of characters.

(function) # ↑

Specs:

  • repeating_xor(iodata, iodata) :: binary

Performs bit-wise XOR (exclusive or) on the data supplied, repeating key as necessary to ensure equal length.

Examples

<<7, 14, 14, 3, 0, 19>>
