chunk(data, n)
(function) # ↑

Specs:

  • chunk(binary, non_neg_integer) :: [binary]

Returns a list containing binaries of size n, where each new chunk starts n bytes into the data.

If there are not enough bytes to fill the final chunk, the partial chunk will be discarded from the result.

Examples

iex> Matasano.Helper.chunk("abcabcabc", 3)
["abc", "abc", "abc"]

iex> Matasano.Helper.chunk("xxxyyyzzz", 4)
["xxxy", "yyzz"]
