pad_pkcs7(message, blocksize)
(function) # ↑

Specs:

  • pad_pkcs7(String.t, non_neg_integer) :: String.t

Pad the message by extending it to the nearest blocksize boundary, appending the number of bytes of padding to the end of the block.

If the original message is a multiple of blocksize, an additional block of bytes with value blocksize is added.

Examples

iex> Matasano.AES.pad_pkcs7("HELLO", 4)
<<72, 69, 76, 76, 79, 3, 3, 3>>
iex> Matasano.AES.pad_pkcs7("HELLO", 5)
<<72, 69, 76, 76, 79, 5, 5, 5, 5, 5>>
